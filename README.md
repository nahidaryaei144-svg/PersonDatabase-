# PersonDatabase-
این پروژه یک سیستم ساده و قابل توسعه برای ثبت، ذخیره و مدیریت اطلاعات اشخاص می‌باشد. هدف از این سیستم، ایجاد یک دیتابیس منظم است که بتوان اطلاعات افراد را به‌صورت دقیق ثبت، جستجو و ویرایش کرد.
فایل index.html را در پوشه public قرار بدهید
برنامه نود جی اس را نسخه قابل حمل node_portable در کنار فایل ها بگذارید بعد فایل اجرایی run app.bat را اجرا کنید
